# Research folder

## Do you need this for a bounty/PoC skill?

**Optional but recommended (small).** Unlike ML or threat-intel repos, this skill does not need a large `research/` corpus. Keep it to **methodology pointers** and **design rationale** so:

- Maintainers remember *why* a gate exists.
- Contributors do not confuse `research/` with the agent instructions (`skills/.../SKILL.md` is canonical for the agent).

If the folder grows, split into dated notes under `research/notes/`—avoid duplicating long content that belongs in `skills/.../references/`.
